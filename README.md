## Installation

To install this package on your computer locally, first download it from `GitHub`:

```
git clone https://github.com/resplab/LEAP.jl
```

Next, install the package:

```
cd LEAP.jl
julia
> import Pkg
> Pkg.add(path="PATH_TO/LEAP.jl")
```
